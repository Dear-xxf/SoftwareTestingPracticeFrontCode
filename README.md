# Software Testing Practice Front Code
Tongji University Software Engineering College

## 初始化

```
npm install
```

### 安装路由

```
npm install vue-router --save
```

### 安装element-ui

```
npm install element-plus --save
```

### 安装element-ui icon

```
npm install @element-plus/icons-vue
```

### 运行

```
npm run serve
```
